# 投稿指南

## 投稿步骤
1. **Fork本仓库**
2. **选择存放路径** 
   - 新建/修改词条文件到对应分类目录
   - 文件名使用英文词条名（如：`quantum_physics.md`）
3. **编写内容** 
   - 使用[投稿模板](投稿模板.md)格式
4. **提交PR** 
   - 目标分支：`main`
   - PR标题格式：`[新稿] 词条名` 或 `[修订] 词条名`
   - 描述栏注明：
     ```markdown
     ### 修改说明
     [简述内容变动]
     
     ### 关联性说明
     [解释新增链接的合理性]
     ```
5. **审核**
   - 编委会将在**14天内**制定一名编辑完成初审
   - 重大修订需配合编辑进行不超过2次的修改迭代，主要内容一般是修正关联词条等；
   - 通过后，编辑将发动编委会动议投票；
   - 投票通过后，合并入main。
